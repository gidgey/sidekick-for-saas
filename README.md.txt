# Sidekick for SaaS — Chrome Extension  
_For When You Really Just Need a Bridget._

Sidekick for SaaS helps consultants, creators, and founders write thoughtful,
authentic comments on X and LinkedIn — in seconds, and in B2B Marketer, Bridget Willard’s signature tone.

Turn smart comments into warm leads all on your own. WINK.

## Features

- Highlight any post text on X / LinkedIn, then generate 2–4 comment ideas
- “Rewrite like Bridget” toggle for a warm, slightly witty consultant voice
- Copy or **Insert** suggestions directly into the reply box (best effort)
- “Test connection” button in Settings to verify your OpenAI API key
- Friendly error messages for quota, model access, and missing API key
- Brand-aware UI in Bridget’s Mango / Banana Mania palette

## Requirements
- You will need to create an API Key with OpenAI. https://platform.openai.com/settings/organization/api-keys
- This requires API Credits with OpenAI. https://platform.openai.com/settings/organization/billing/overview
